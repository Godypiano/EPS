Audit Trail

Audit Trail

Audit trail keeps track of everything done in the system. Below are the categories of audit trail seen by administrator

  • Axles
  • Axles Overload
  • GVM Overload
  • Categories
  • Departments
  • Permit Days
  • Permit Fees
  • TANESCO Height
  • Permit Vehicles
  • Escort Vehicles
  • Permit Details
  • Permit Invoice
  • Permit Payment

Permit Days

This shows history of actions done in        Permit Days

To see Audit Permit Days records follow the following steps

  1. Select year
  2. Select month
  3. Click submit button to see results


Administrator will be able to see the following details

  • AuditID        
  • Action        
    • I => Insert,
    • U =>Update,
    • D => Delete
  • SNO        
  • Permit Days        
  • Audit Date        
  • Audit By

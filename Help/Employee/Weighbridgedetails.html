Weighbridge details

Weighbridge details

To go to Weighbridge details screen, Click Weighbridge details link under setup menu as indicated below


Create Weighbridge details

 

  1. Enter Weighbridge ID
  2. Enter Weighbridge details
  3. Select Weighbridge details
  4. Click Create button to save


Update Weighbridge details

  1. Select Weighbridge details from the list
  2. Edit Weighbridge
  3. Click Update button to save


Delete Weighbridge details

  1. Select Weighbridge details  from the list
  2. Click Delete button
